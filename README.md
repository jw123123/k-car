국내 최대 직영중고차 플랫폼 K Car의 회사 소개 사이트를 리디자인한 프로젝트입니다. 중고차 거래 자체가 자원 재사용과 순환 소비라는 ESG 가치와 연결되지만 기존 사이트에서는 이 점이 충분히 드러나지 않아, 콘텐츠 구조와 시각적 표현, 인터랙션 요소를 개선하여 브랜드 경험을 보여주는 것을 목표로 제작하였습니다.

🔗 배포 사이트: [https://jw123123.github.io/k-car/](https://jw123123.github.io/k-car/)

---

## 📋 프로젝트 개요

### 프로젝트 목적

기존 K Car 홈페이지는 기업 정보 중심의 단조로운 레이아웃으로 구성되어 있어 브랜드 경쟁력이 직관적으로 전달되지 않았고, 텍스트와 정적인 이미지 위주의 콘텐츠 구성으로 사용자 몰입도가 낮았습니다. 또한 서비스 정보와 CTA가 분산되어 있어 사용자 행동 유도가 부족한 한계가 있었습니다.

이에 브랜드 아이덴티티를 강조한 비주얼과 핵심 메시지 중심의 메인 화면으로 개선하고, 스크롤 인터랙션과 모션 효과를 활용하여 콘텐츠 탐색 경험과 몰입감을 높였습니다. 서비스 특징은 카드형 UI로 재구성하고 명확한 CTA를 배치하여 사용성과 전환율을 개선하였습니다.

### 프로젝트 목표

기업의 경영철학을 직관적으로 전달하고 기술역량을 포함한 핵심 서비스를 소개하는 것을 목표로 하였습니다.

### 프로젝트 타겟

자동차에 관심이 많은 성인을 대상으로 하며, 경제성과 신뢰를 우선하는 현실 소비층이자 차량 구매 시 가격뿐 아니라 품질과 정보 투명성을 중시하는 사용자입니다. 핵심 니즈는 신뢰할 수 있는 차량·기업 정보 확인, 합리적이면서 가치 있는 소비라는 확신, 직관적인 메뉴와 빠른 정보 탐색입니다.

### 프로젝트 방식

- 팀 프로젝트 (3인, 기여도 50%)
- Figma를 활용한 UI 설계
- HTML/CSS/JavaScript 기반 퍼블리싱
- Notion을 활용한 기획 및 회의록 관리
- Git을 통한 버전 관리

## ✨ 주요 기능

- 브랜드 → 사업분야 → ESG로 이어지는 정보 흐름의 메인 화면 스크롤 인터랙션 구현
- 영상 비주얼과 핵심 메시지 중심의 히어로 영역 구성
- 서비스(국내 최대 직영 시스템 / 내차팔기 홈서비스 / 품질 점검 / 내차사기 홈서비스 / 책임환불제) 카드형 UI 구현
- 지속가능경영(ESG) 영역의 Environment / Social / Governance 탭 전환 및 데이터 시각화
- NEWS 영역 카드 슬라이더 구현
- 인재채용 영역 카드형 UI 및 CTA 구성
- 전체 메뉴 구조를 한 화면에 노출하는 사이트맵 구현
- PC / Tablet / Mobile 반응형 대응 (12 / 8 / 5 컬럼 그리드 기준)
